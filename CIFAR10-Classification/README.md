# CIFAR-10 이미지 분류 모델 개발 및 분석

본 프로젝트는 컴퓨터 비전의 기초인 이미지 분류 문제를 해결하기 위해 **CIFAR-10 데이터셋**을 활용하여 딥러닝 모델을 설계하고, 다양한 최적화 기법을 통해 성능을 극대화하는 과정을 담고 있습니다.

## 1. 프로젝트 요약(Project Summary)

- **목표**: CIFAR-10 데이터셋에서 80% 이상의 정확도를 달성하는 효율적인 딥러닝 모델 설계.
- **주요 성과**: 테스트 데이터 기준 81.61%의 정확도 달성.
- **모델 효율성**: 약 650만개 파라미터로 높은 성능과 자원 효율성 구현.
- **핵심 기법**: CNN(LeNet-5 기반), 드롭아웃(Dropout), L2 정규화, 배치 정규화(Batch Normalization).

## 2. 데이터셋 정보(Dataset Overview)

-   **구성:** 10개 클래스(비행기, 자동차, 새 등), 총 60,000개의 32x32 컬러 이미지.
-   **데이터 분할:** 훈련 데이터 50,000개, 테스트 데이터 10,000개.
-   **전처리:** 픽셀 값 정규화(0~1 범위), 원-핫 인코딩(One-Hot Encoding), 훈련-검증 데이터 분리(8:2).

## 3. 모델 발전 과정 (Model Evolution)

성능 향상을 위해 단순 신경망에서 심층 신경망, 합성곱 신경망으로 점진적으로 확장하며 실험을 진행했습니다.

1.  **모델 1 (ANN):** 단층 인공 신경망으로 시작했으나 이미지 특성을 살리지 못해 매우 낮은 성능 기록.
    
2.  **모델 2 (DNN):** 레이어 추가 및 RMSProp 옵티마이저 적용으로 정확도 약 45% 달성 (모델 1 대비 약 20% 향상).
    
3.  **모델 3 (CNN):** LeNet-5 구조를 참고한 합성곱 신경망으로 전환하여 공간 정보 학습, 정확도 40%p 이상 급증.
    
4.  **최종 모델 (Optimized CNN):** 과대적합 방지를 위해 드롭아웃(계수 0.5), L2 정규화(0.0001), 배치 정규화를 순차적으로 적용하여 안정적인 학습 및 일반화 성능 확보.

## 4. 성능 분석 (Results & Analysis)

-   **정확도:** 테스트 데이터셋에서 **81.61%** 를 기록하여 초기 목표인 80%를 초과 달성함.
    
-   **혼동 행렬(Confusion Matrix) 분석:**
    
    -   자동차, 버스, 트럭 등 형태가 명확한 클래스에서 높은 탐지 정확도 확인.
        
    -   개와 고양이 클래스 간 혼동 비율이 높게 나타남(유사한 외형 및 낮은 해상도 기인).
        
-   **지표:** PR-Curve 및 ROC-AUC 분석을 통해 전반적으로 고른 성능 유지 확인.
    

## 5. 향후 개선 방안 (Future Work)

-   **모델 고도화:** 레이어 수 확장을 통해 고수준 특징 추출 능력 강화.
    
-   **데이터 증강 최적화:** 개와 고양이처럼 혼동이 잦은 클래스를 위해 특화된 증강 전략 도입.
    
-   **확장성:** CIFAR-100과 같은 더 복잡한 대규모 데이터셋으로의 확장 가능성 탐구.
