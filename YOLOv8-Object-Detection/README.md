# YOLOv8 기반 도로 데이터 객체 탐지 모델 설계 및 분석

본 프로젝트는 도로 환경에서 주행 중 발생할 수 있는 다양한 객체를 실시간으로 식별하고 위치를 추정하는 **Object Detection** 과제를 수행했습니다. 데이터 정제부터 전이 학습을 통한 모델 최적화까지의 전 과정을 포함합니다.

## 1. 프로젝트 요약 (Project Summary)

-   **목표**: 도로 환경의 주요 객체(자동차, 보행자 등)를 탐지하고 바운딩 박스로 시각화하여 높은 정밀도와 재현율 확보.
    
-   **핵심 성과**: 전이 학습 적용 후 **mAP@50 지표가 약 9,860% 향상**되는 비약적 성능 개선 달성.
    
-   **최종 지표**: mAP@50 **0.7391**, mAP@50-95 **0.5542** 기록.
    
-   **모델**: YOLOv8n (Nano) 모델 활용 (실시간 처리 및 경량화 특화).
    

## 2. 데이터셋 및 전처리 (Dataset & Preprocessing)

-   **데이터 구성**: 도로 주행 이미지 11,205장 및 어노테이션 10,009개.
    
-   **클러스 정보**: Automobile, Bus, Truck, Pedestrian 등 총 11개 클래스.
    
-   **엔지니어링 포인트**:
    
    -   **데이터 정제**: 이미지와 어노테이션 간의 이름 불일치 파일을 전수 조사하여 삭제 및 매칭 (총 10,009개 세트 확보).
        
    -   **형식 변환**: JSON 포맷의 바운딩 박스 좌표를 YOLO 학습에 적합한 `[class_id, x_center, y_center, width, height]` 정규화 형식으로 변환.
        
    -   **데이터 분할**: 훈련(70%), 검증(20%), 테스트(10%) 세트로 분리하여 학습 안정성 확보.
        

## 3. 모델 학습 전략 (Training Strategy)

-   **전이 학습 (Transfer Learning)**: COCO 데이터셋으로 사전 학습된 가중치를 초기값으로 사용하여 초기 성능 및 학습 속도 개선.
    
-   **학습 환경**:
    
    -   **GPU**: NVIDIA A100 40GB 활용.
        
    -   **하이퍼파라미터**: Epochs 100, Batch size 64, Optimizer Adam, Image size 640x640.
        

## 4. 성능 분석 (Results & Analysis)

-   **학습 안정성**: 훈련 데이터와 검증 데이터 간의 손실(Loss) 값이 유사하게 감소하여 **과대적합(Overfitting) 없이 안정적으로 학습**되었음을 확인.
    
-   **클래스별 성능**:
    
    -   자동차, 버스, 트럭 등 형태가 명확한 클래스에서 높은 탐지 성능 기록.
        
    -   외형이 유사한 **자전거-오토바이**, **일반 버스-스쿨버스** 간의 일부 혼동 발생 확인 (향후 개선 과제).
        
-   **효과 검증**: 무작위 초기화 모델 대비 학습 시간 단축 및 초기 단계부터 높은 신뢰도 확보.
    

## 5. 향후 개선 방향 (Future Work)

-   **소수 클래스 보완**: 빈도수가 낮은 클래스(자전거 등)의 성능 향상을 위한 데이터 증강(Augmentation) 적용.
    
-   **미세 조정 (Fine-tuning)**: 도로 환경 특화 레이어 추가 및 하이퍼파라미터 튜닝을 통한 정밀도 고도화.
